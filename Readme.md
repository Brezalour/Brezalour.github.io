# 🎮 Jeu du Pendu - Version Web

Un jeu du pendu interactif et moderne en JavaScript, HTML et CSS.  
Avec animations, effets sonores, clavier virtuel et mode clair/foncé.

---

## 🚀 **Fonctionnalités**
✔ Sélection de **trois niveaux de difficulté** (Facile, Normal, Difficile).  
✔ **Clavier virtuel interactif** pour jouer à la souris.  
✔ **Effets sonores dynamiques** pour les lettres correctes, erreurs, victoires et défaites.  
✔ **Animations fluides** (lettres qui pop, pendu qui secoue, victoire stylisée).  
✔ **Mode clair / sombre** 🌙 pour un meilleur confort visuel.  
✔ **Système de progression visuelle** avec une barre d’erreur interactive.  

---

## 🛠 **Technologies utilisées**
- **HTML5** → Structure du jeu  
- **CSS3** → Design et styles modernes  
- **JavaScript (ES6+)** → Logique du jeu  
- **GSAP** → Animations avancées

2. **Ouvre le fichier index.html dans un navigateur.**
   
 **🎮 Comment jouer ?**
**Choisis un niveau de difficulté (Facile, Normal, Difficile).**
**Devine le mot en tapant des lettres sur le clavier ou en utilisant le clavier virtuel.**
**Chaque erreur affiche une partie du pendu (selon la difficulté choisie).**
**Trouve le mot avant d’atteindre la limite d’erreurs !**
**Utilise le bouton "Rejouer" pour recommencer.**
